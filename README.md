# Jogo da Cobra

Um clássico Snake Game feito em Python com Pygame... porém com algumas diferenças.

## Features

* Comida = logo do Python
* Rotação da cabeça dependendo da direção
* Sistema de pontuação
* Game Over
* Código 100% acadêmico e definitivamente aprovado pela faculdade 👍

## 🚀 Tecnologias

* Python
* Pygame

## ▶️ Como rodar

Instale o pygame:

```bash
pip install pygame
```

Depois execute:

```bash
python main.py
```

## 📁 Arquivos necessários

Deixe tudo na mesma pasta:

```plaintext
main.py
professor.png
python_food.png
```

## 🎮 Controles

* ↑ ↓ ← → = mover
* ENTER = reiniciar
* ESC = sair
